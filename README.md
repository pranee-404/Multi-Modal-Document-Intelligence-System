# Multi-Modal-Document-Intelligence-System
Built a production-ready AI system that combines computer vision and LLMs to extract, reason over, and validate text, tables, and visual elements from scanned and structured documents using tools like: Python, OpenCV, YOLO/DETR, OCR (Tesseract/EasyOCR), Multi-Modal LLMs (GPT-4V/Claude/LLaVA), LangGraph, Chroma, FastAPI, Docker.
